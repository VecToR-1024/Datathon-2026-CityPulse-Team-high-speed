CityPulse is a data analysis and visualization project designed to address urban accessibility issues in Seattle.

The project is based on the dataset sourced from Harvard Dataverse: https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/YOTY6A, and was created for the 7th UW Datathon.

Access the webpage [here](https://vector-1024.github.io/Datathon-2026-CityPulse-Team-high-speed_machinery/) to see the full presentation.
