CityPulse

Cities shape how people move, navigate, and participate in everyday life—but the ability to move freely is not equally distributed. Sidewalk design, curb ramps, surface quality, and obstacles can create friction that affects safety, independence, and belonging for many people.

Our Work

Government Priority Index (GPI)

Indicates the urgency level of each accessibility barrier to guide government repair priorities.

Individual Accessibility Index (IAI)

Serves as a guide for citizens and travelers, especially those encountering accessibility difficulties, to choose better routes when traveling.

Distance-Weighted Radius Neighbor Regression Model

Estimates indexes for any given location in Seattle using spatial analysis techniques.

Heatmaps are created to represent the analysis visually, where lighter regions are associated with more severe accessibility issues, and darker regions indicate less severe accessibility problems.

Model Validation

Model Performance

Our Distance-Weighted Radius Neighbor Regression model successfully captures spatial accessibility patterns:

  • The estimated heatmaps closely match the original data distribution
  • Hot spots (high-friction areas) are consistently identified in both original and predicted maps
  • The model reveals data sparsity—showing areas where no data exists within the search radius
  • Smooth interpolation provides actionable insights for unmeasured locations
  • However, the model slightly deviates from the real-world situation when fewer samples are presented around
